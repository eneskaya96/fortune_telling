# fortune_telling

To run with Chrome from terminal 
        
    flutter run -d chrome

    flutter run --release 

for icon

add this to .yaml

    flutter_icons:
    android: "launcher_icon"
    ios: true
    image_path: "assets/icon/icon.png"

then run this 

    flutter pub run flutter_launcher_icons
